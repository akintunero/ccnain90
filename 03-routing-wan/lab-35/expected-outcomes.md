## Expected Outcomes – Lab 21 Routing and WAN Edge

By completing this lab, learners should be able to:

1. **Integrate LAN and WAN routing**
   - Explain how inter-VLAN routing and WAN edge routing interact.
   - Describe the path traffic takes from a VLAN host to an external ISP host.
2. **Implement point-to-point WAN addressing**
   - Use a /30 network from the WAN blueprint (`10.255.X.0/30`) correctly.
   - Avoid overlapping or misaligned address assignments.
3. **Configure and validate static routing**
   - Implement static routes on HQ and ISP routers.
   - Confirm that routing tables reflect the expected prefixes.
4. **Troubleshoot common LAN–WAN issues**
   - Identify causes of one-way connectivity or missing routes.
   - Use `show ip route`, `ping`, and `traceroute` effectively.

These outcomes lay the groundwork for more advanced dynamic routing and multi-site WAN designs in subsequent labs.
