## Expected Outcomes – Lab 01 Foundations

By completing this lab, learners should be able to:

1. **Design a small office IPv4 addressing plan**
   - Allocate subnets from the `192.168.X.0/24` small-network blueprint.
   - Reserve gateway addresses and host ranges logically and consistently.
2. **Implement basic VLAN segmentation**
   - Create user and management VLANs using the standard VLAN ID plan.
   - Correctly assign access ports and trunk links on the access switch.
3. **Configure default gateways and basic inter-VLAN routing**
   - Configure Layer 3 interfaces (SVIs or subinterfaces) with correct IPs and masks.
   - Verify connectivity between VLANs and towards a simulated ISP network.
4. **Apply CCNA-level device management practices**
   - Configure hostnames, secure management access, and local user accounts.
   - Use SSH instead of Telnet for remote administration where supported.
5. **Systematically verify and troubleshoot**
   - Use show and ping commands to validate VLAN, interface, and routing status.
   - Identify and correct common misconfigurations in VLAN and gateway setup.

Learners should be comfortable repeating this foundational build in future labs as a baseline for more advanced topics.
